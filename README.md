# Mountrip_Adventures_Analysis
End-to-End Business Analytics Project using Python, SQL, and Power BI



Overview:-

End-to-end analytics project to analyze travel bookings, revenue, cost, and profit.
Built using Python for data cleaning, SQL for modeling, and Power BI for visualization.

Tech Stack:-

*Python (Pandas)
*MySQL
*Power BI


Workflow:-

Python → Data cleaning & preprocessing
SQL → Data modeling & queries
Power BI → Dashboard & insights


Dashboard Includes:-

*Revenue, Cost, Profit, Margin (KPIs)
*Revenue by Package
*Bookings by Platform
*Monthly Trend
*Booking Type (Solo vs Group)
*Cost Breakdown


Key Insights:-

*Kedarnath drives majority of bookings
*Group bookings generate higher revenue
*Costs vary by package and logistics
